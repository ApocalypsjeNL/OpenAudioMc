Welcome to OpenAudioMc

  • Donations and Funding

    The OpenAudioMc project is entirely run by myself (Mindgamesnl), which also means that I have to cover all costs related to hosting, streaming, fees and maintenance. Support in the form of donations are really appreciated and help the project stay around.

  • Craftmend Account

    Craftmend Accounts are the portal where server managers, admins and developers can configure and manage their OpenAudioMc installations, easily customize their client without any coding, or access the rest API's and addons if you're more into that stuff.

  • Discord

    The OpenAudioMc Discord server is a great place to talk about the project, ask for support, communicate with other users and project developers and to stay up to date with the growth of the project. Feature development, public testing phases and feedback pools are usually done through there.

  • Development

    The core OpenAudioMc plugin, documentation, specs and web-client are maintained in a public Github repository, where we encourage pull requests if you find something that you want to have improved.

  • API's and Technical Notes

    OpenAudioMc comes with an easy to use, cross platform rest and Java API that enable you to hook into almost all features, control data flow and do pretty much whatever you want. Its a great place to start when you're planning to integrate OpenAudioMc into your existing server, or to discover possibilities when planning your features/future expansion.

  • Documentation and Support

    The Documentation is a great place to start when learning how to use OpenAudioMc, it contains all the information that an admin or developer might need to make the most of our platform.
